# WithBand

バンド活動向けのソーシャル掲示板サイトです。メンバー募集、ライブ告知、セッション募集、機材売買を同じ画面で横断できる構成にしています。

## セットアップ

```bash
npm install
```

## 開発サーバー

```bash
npm run dev
```

## 現在の実装内容

- 投稿検索とカテゴリ絞り込み
- 地域別の掲示板フィード
- イベント告知カード
- コミュニティ参加導線

## 利用可能なスクリプト

- `npm run dev`: Turbopack で開発サーバーを起動
- `npm run build`: 本番ビルドを作成
- `npm run start`: ビルド済みアプリを起動
- `npm run lint`: ESLint を実行
